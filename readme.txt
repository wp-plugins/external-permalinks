This plugin allows you to create posts with external permalinks. Great for guest posts or cross promoting your blog posts.

== Description ==

You can use this plugin to link directly to external posts. Possible usages: 

* Cross promote posts across blogs.
* Link directly to the guest post at another blog.
* Etc, etc..

**More info**

* Check out [External Permalinks](http://www.improvingtheweb.com/wordpress-plugins/external-permalinks/) for more information.
* Check out my other [Wordpress plugins](http://www.improvingtheweb.com/).

== Installation ==

1. Download the plugin.
2. Unzip the plugin and upload the directory to `wp-content/plugins`.
3. Activate the plugin.
4. If necessary, go to the "External Permalink" options page and make your changes.
5. When adding/editing posts, enter the external URL in the field labeled "External URL", typically at the bottom of the screen.
